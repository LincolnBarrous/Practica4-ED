Descripción detallada

template<class T, class U>
class Diccionario< T, U >

T.D.A Diccionario.

Una instancia d del tipo de dato abstracto Diccionario sobre un dominio T y U es una lista de datos de los definidos anteriormente. En un diccionario, las entradas se ordenan en función de la clave, de tipo T.

Se implementa mediante una lista de elementos de tipo data, que son parejas de tipo T y U
